## The Full Picture — All Builder Types in One Place

```

WebApplicationBuilder (builder)
│
├── builder.Configuration  →  IConfigurationBuilder
│   PURPOSE: Assemble config sources
│   USE FOR: appsettings, env vars, Secrets Manager, Parameter Store
│   TIMING:  Before builder.Build()
│
├── builder.Services       →  IServiceCollection
│   PURPOSE: Register DI services
│   USE FOR: AddControllers, AddOptions, AddSingleton, AddHttpClient
│   TIMING:  Before builder.Build()
│             but AFTER config sources are added
│
├── builder.Host           →  IHostBuilder
│   PURPOSE: Configure host-level concerns
│   USE FOR: Serilog, host lifetime, background services
│   TIMING:  Before builder.Build()
│
└── builder.Build() returns WebApplication (app)
    │
    └── app  →  IApplicationBuilder
        PURPOSE: Configure HTTP request pipeline
        USE FOR: UseAuthentication, UseAuthorization,
                 UseHealthChecks, UseMiddleware
        TIMING:  After builder.Build()
                 Order of calls = order requests are processed

```
